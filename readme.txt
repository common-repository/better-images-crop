Better Images Crop made croping results same as width and height in media settings

== Description ==

Better Images Crop is plugin that hook in to cropping proccess.
At default, when you upload an image to your site, you`ll got a thumbnail and rescale images.

Using this plugin, except the source image, other media max limit will change into fixed thumbnail images.


== Installation ==

You can download and install the Better Image Crop Plugin using the built in WordPress plugin installer. If you download this Plugin manually, make sure it is uploaded to "/wp-content/plugins/better-images-crop/".


== Frequently Asked Questions ==

= If after installing this plugin, will all images that i have will change? =

No, only images after this plugin activate

= Can i change cropping area? =

Yes, you can. In this plugin user can set their vertical and horizontal area that want to make as thumbail.

= Can each user member in my site have their own settings? =

No
